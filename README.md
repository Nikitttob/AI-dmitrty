# 🏠 Ассистент ЖКХ + Право

RAG-ассистент по правовой системе РФ, жилищно-коммунальному хозяйству и банковскому обслуживанию фондов капитального ремонта.

## Что это?

AI-ассистент с встроенной базой знаний из 103 вопросов-ответов:
- **76 вопросов** по правовой системе РФ и ЖКХ
- **27 вопросов** по банковскому обслуживанию фондов капремонта

Ассистент находит релевантный контекст в базе знаний и отправляет его вместе с вопросом в Claude API для формирования точного ответа со ссылками на нормативные акты.

## Быстрый старт

### 1. Установите Node.js

Скачайте и установите с [nodejs.org](https://nodejs.org/) (версия 18+).

### 2. Установите зависимости

```bash
cd zhkh-assistant
npm install
```

### 3. Запустите (два варианта)

**Вариант А — С Claude API (рекомендуется):**

```bash
# Windows (PowerShell):
$env:ANTHROPIC_API_KEY="sk-ant-api03-ваш-ключ"
node server.js

# Windows (CMD):
set ANTHROPIC_API_KEY=sk-ant-api03-ваш-ключ
node server.js

# macOS / Linux:
export ANTHROPIC_API_KEY=sk-ant-api03-ваш-ключ
node server.js
```

**Вариант Б — Без API ключа (оффлайн-режим):**

```bash
node server.js
```

В оффлайн-режиме ассистент отвечает напрямую из базы знаний (без генерации Claude), но всё равно полезен для поиска информации.

### 4. Откройте в браузере

```
http://localhost:3000
```

## Как получить API ключ

1. Зарегистрируйтесь на [console.anthropic.com](https://console.anthropic.com)
2. Перейдите в Settings → API Keys
3. Создайте новый ключ (начинается с `sk-ant-...`)
4. Используйте как описано выше

## Как это работает (RAG)

1. **Пользователь** задаёт вопрос
2. **Поисковый движок** находит 5 наиболее релевантных пар «вопрос-ответ» из базы знаний (keyword matching с partial match)
3. **Контекст** отправляется вместе с вопросом в Claude Sonnet API
4. **Claude** формирует ответ, опираясь на найденный контекст и ссылаясь на конкретные статьи законов
5. **Интерфейс** показывает ответ + раскрываемый список использованных источников

## Расширение базы знаний

Чтобы добавить новые вопросы, отредактируйте `data/knowledge_base.json`:

```json
{
  "q": "Ваш вопрос?",
  "a": "Ответ на вопрос с подробностями...",
  "r": "Ссылка на нормативный акт",
  "t": "l"
}
```

Где `t` — тип: `"l"` для права/ЖКХ, `"b"` для банков/капремонта.
